# yongmusic
> 自研自用的android音乐播放器

### 为什么开发
- 为新技术研究提供一个demo
- 高频自用，不足的功能有动力开发

### 已有功能
- 本地播放
- 网络搜索
- 播放列表
- 歌单、专辑、歌手管理
- 最近播放

### 待开发
- 歌词
- 通知
- 锁屏展示
- 导入音乐
- 下载管理
